Serial port logger through [COM ports](https://en.wikipedia.org/wiki/COM_(hardware_interface)).

* Doesn't require `std` (`no_std`)
* Doesn't require `alloc`

```rust,no_run
use log::*;

fn main() {
    com_logger::init();

    info!("Starting");
}
```

## Configuration

```rust,no_run
use log::*;

fn main() {
    com_logger::builder()
        .base(0x2f8)                  // Use COM2 port
        .filter(LevelFilter::Debug)   // Print debug log
        .setup();

    debug!("Hello");
}
```
